PolyDeck - Vocabulary Learning App

PolyDeck is a mobile application that helps users learn English vocabulary effectively through flashcards and quizzes.

 Technologies
- Java (Android)
- Firebase
- MongoDB

 Features
- Learn vocabulary using Flashcards
- Quiz system to test knowledge
- Ranking system
- User authentication
- Manage vocabulary topics

 How to run
1. Clone this repository
2. Open in Android Studio
3. Run the app on emulator or device
